Class ActorSystem

ActorSystem.

The system treats all actors equally ignoring implementation details instead leaving that to the create function of their Templates.

Actors are stored in the internal actors table and should never be modified directly. This class provides all the methods needed for actor implementations to interact with the system.

Each implementation could be seen as a sort of "driver" that hooks into a "kernel" implementation to provide functionality to an application.
